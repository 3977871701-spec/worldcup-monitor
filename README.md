# ⚽ 2026 FIFA 世界杯智能监控与预测系统

> 基于泊松分布蒙特卡洛模拟 + 贝叶斯自学习的全自动世界杯比赛预测与实时监控平台

## 📋 项目简介

本系统是一套面向 2026 年美加墨世界杯（FIFA World Cup 2026）的全栈智能监控与预测系统。系统从 ESPN 实时 API 抓取比赛数据，运用 **泊松分布蒙特卡洛模拟**（20万次/场）进行赛前预测，结合 **贝叶斯自学习引擎** 在赛后自动校准模型参数，并提供 **实时比分推送**、**体彩赔率监控**、**赛后结算与准确率追踪**、**实时比赛胜率动态计算** 等完整功能。

系统采用纯 Node.js 后端 + 原生 HTML/CSS/JS 前端的轻量架构，无需数据库和构建工具，开箱即用。

---

## 🚀 功能特性

### 1. 🎲 泊松分布蒙特卡洛模拟（Poisson Monte Carlo Simulation）
- 每场比赛执行 **20万次** 蒙特卡洛模拟
- 基于泊松分布采样进球数，计算主胜/平局/客胜概率
- 综合考虑球队排名差、近期状态、世界杯历史表现、球员因素（适应度/伤病/经验）
- 动态排名权重：排名差距越大，权重越高（最高 1.8 倍）
- 输出：预测比分、胜率分布、Top 5 比分概率、xG 期望进球值

### 2. 🧠 贝叶斯自学习引擎（Bayesian Auto-Learning）
- 每场比赛结束后自动触发学习流程
- 分析方向准确率与比分准确率，动态调整模型参数：
  - `rankWeight`：排名权重（准确率低则降低，高则提升）
  - `baseGoals`：基础期望进球（对比实际/预测进球偏差校准）
  - `homeBonus`：主场优势系数
  - `drawBoost`：平局概率修正（解决平局预测不足问题）
- 自动更新球队 `motivation_factor`（动力因子）
- 学习结果写入 `data/team_profiles.json`，并重新预测所有未开始比赛

### 3. ⚡ 24/7 全天候预测系统（24/7 Prediction Engine）
- `predict-24-7.js` 作为独立守护进程持续运行
- 每 5 分钟自动获取 ESPN 赛程，预测接下来的 2 场比赛
- 已锁定比赛（已开始/已结束）不会被覆盖
- 自动写入 `data/predictions.json` 供前端展示

### 4. 📡 实时比分推送（Real-time Score via SSE）
- 服务端通过 **Server-Sent Events (SSE)** 实时推送比赛数据到前端
- 智能更新频率：比赛进行中 **每 5 秒** 刷新，休赛期每 30 秒
- 进球时前端弹出 **进球通知弹窗**，高亮动画效果
- 支持多客户端同时连接

### 5. ⚡ 实时比赛动态预测（Live Match Dynamic Prediction）
- `realtime.html` 提供比赛进行中的实时胜率计算
- 综合因素动态调整：
  - 当前比分差（随时间递增影响权重）
  - 比赛时间（平局时越接近终场平局概率越高）
  - 红牌事件（每张红牌影响约 15% 概率）
  - 控球率、射正数、角球等实时统计
- 预测最终比分，展示比赛关键事件时间线
- 通过 SSE 实现 30 秒自动刷新

### 6. 📊 赛后结算与准确率追踪（Post-match Settlement）
- `settle.js` 在比赛结束后自动执行结算
- 与赛前预测对比，判定：
  - ✅ 方向正确（预测胜/平/负方向命中）
  - ✅ 比分正确（精确比分命中）
  - ⚠️ 冷门判定（实际结果对应概率 < 30%）
- 综合准确率评分（0-100 分）
- 每 5 分钟由 `server.js` 自动调度执行

### 7. 💰 体彩赔率监控（Odds Monitoring）
- `odds-monitor.js` 监控体彩（sporttery.cn）赔率数据
- 自动移除已结束比赛的赔率记录
- 赔率数据存储在 `data/detailed_odds.json`
- 前端展示赔率与隐含概率换算

### 8. 🔮 七层全维度分析框架（7-Layer Analysis — analyzer.py）
- Python 分析脚本，提供七层深度分析：
  1. **基础数据**：积分、首轮进球/失球、赔率
  2. **球队实力**：FIFA 排名、战术风格、优势/劣势
  3. **战术匹配**：控球型 vs 反击型等风格对抗分析
  4. **心理因素**：生死战压力、出线形势、触底反弹
  5. **隐藏因素**：球员适应度、伤病影响
  6. **庄家思维**：赔率隐含概率计算
  7. **历史规律**：验证案例库

### 9. 📺 直播转播导航
- 集成中国大陆主流转播平台链接：
  - CCTV5 体育频道（免费全场次）
  - 咪咕视频、懂球帝、直播吧、腾讯体育、88看球
  - FOX Sports、Peacock（英文/西语）

### 10. 🏆 积分榜与赛程管理
- 小组积分榜自动生成（胜/平/负/进/失/净胜球/积分）
- 赛程浏览与搜索（支持中英文球队名）
- 搜索结果导出功能

### 11. 🌐 完整中文化
- 32 支参赛球队中文名映射
- 球员姓名中文翻译（梅西、姆巴佩、哈兰德等）
- 球场名称中文翻译（SoFi 体育场、大都会人寿体育场等）
- 比赛事件翻译（进球、红牌、黄牌、换人、角球等）

## 🎮 使用方法

### 启动主服务器

```bash
node server.js
```

服务器将在 `http://localhost:8080` 启动，自动完成以下操作：
1. 加载已有预测数据
2. 从 ESPN API 获取最新比赛数据
3. 执行赛后结算与学习
4. 启动智能更新调度（比赛中 5 秒 / 休赛 30 秒）
5. 每 5 分钟自动执行结算 + 学习循环

### 启动 24/7 预测守护进程

```bash
node predict-24-7.js
```

独立运行的预测引擎，每 5 分钟：
1. 从 ESPN API 获取赛程
2. 找到接下来 2 场未开始的比赛
3. 执行 20 万次蒙特卡洛模拟
4. 写入 `data/predictions.json`

### 手动执行赛后结算

```bash
node settle.js
```

获取已结束比赛，对比预测结果，计算准确率，并自动触发学习。

### 手动执行模型学习

```bash
node learn.js
```

从结算记录中学习，调整模型参数，重新预测所有未开始比赛。

### 运行 Python 分析

```bash
python3 analyzer.py
```

对指定比赛执行七层全维度分析，输出分析报告。

### 监控赔率

```bash
node odds-monitor.js
```

清理已结束比赛的赔率数据，保持赔率数据时效性。

## 🧮 预测模型原理

### 泊松分布模拟

每场比赛的期望进球（xG）计算公式：

```
homeXG = 1.3 × (主队场均进球/1.2) × (1.2/客队场均失球) × (1 + 近期状态×0.2) × (1 + 世界杯表现×0.1) × 球员因子 × 1.15(主场)
awayXG = 1.3 × (客队场均进球/1.2) × (1.2/主队场均失球) × (1 + 近期状态×0.2) × (1 + 世界杯表现×0.1) × 球员因子
```

球员因子 = 适应度 × (1 - 伤病影响×0.5) × (0.7 + 经验×0.3)

### 贝叶斯自学习参数

| 参数 | 默认值 | 调整逻辑 |
|------|--------|----------|
| `rankWeight` | 0.015 | 方向准确率 <50% → 0.012; >70% → 0.018 |
| `baseGoals` | 1.3 | 实际进球偏差 ±0.3 时微调 |
| `homeBonus` | 0 | 主场预测准确率 <40% 时降至 -0.05 |
| `drawBoost` | 0 | 实际平局率 >20% 但预测 <10% 时提升 |
